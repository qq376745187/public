跨境合规体检助手 隐私政策

最后更新：2025年6月1日

1. 我们收集什么数据

不收集。本扩展不收集、不上传、不存储任何用户个人信息或商品数据。我们没有任何后端服务器。

2. 用户输入的数据如何处理

  • 用户粘贴的文本和上传的图片仅在本地浏览器中处理。
  • 文本违禁词扫描完全在本地完成，基于浏览器内置的违禁词库进行匹配，不发送到任何服务器。
  • 图片侵权检测和AI 深度分析、AI 替换文案生成功能需要调用 Google Gemini API，相关数据（文本内容、图片数据）会发送到 Google 服务器进行分析。
  • 分析完成后，数据不会被本扩展保存，也不会发送到我们的服务器（因为我们没有服务器）。
  • 用户可随时通过「清空」按钮清除当前检测内容，或通过卸载扩展清除所有本地数据。

3. API Key 的处理

  • 用户自行输入的 Gemini API Key 存储在浏览器的 chrome.storage.local 中，仅保存在用户本地设备。
  • API Key 仅在本扩展内部用于调用 Google Gemini API，不会发送到任何第三方。
  • 用户可随时在扩展设置中修改或清除 API Key。

4. 第三方服务

  • Google Gemini API：仅在用户使用图片检测、AI 深度分析或 AI 替换文案功能时调用。用户可在 Google 隐私政策中了解其数据处理方式：https://policies.google.com/privacy
  • 除上述服务外，本扩展不接入任何其他第三方服务，不进行任何数据埋点、统计分析或行为追踪。

5. 权限说明

  • sidePanel：提供侧边栏操作界面。
  • storage：在本地存储用户设置（API Key、所选平台）和违禁词库缓存。
  • host_permissions（generativelanguage.googleapis.com）：调用 Gemini API 进行图片侵权分析与文案处理。
  • optional_host_permissions（按需申请）：仅当用户主动添加图片 URL 进行检测时，才会申请访问对应图片地址的权限，用于读取图片数据。本扩展不主动访问任何网页。

6. 数据安全

  • 本扩展不采集任何平台商品数据，不注入网页读取内容，法律风险最低。
  • 所有本地存储的数据均在用户设备上，用户拥有完全控制权。
  • API Key 通过浏览器本地存储保管，不上传至任何中间服务器。

7. 数据删除

所有数据存储在用户本地浏览器中，用户可随时通过以下方式删除：

  • 点击扩展内「清空」按钮清除当前检测内容；
  • 在扩展设置中清除 API Key；
  • 卸载扩展；
  • 清除浏览器数据。

8. 儿童隐私

本扩展面向跨境电商卖家等成年专业人士，不面向 18 岁以下未成年人。我们不会故意收集任何未成年人的个人信息。

9. 免责声明

本扩展提供的合规检测结果由 AI 辅助生成，仅供参考，不构成法律建议。实际上架合规性以平台审核结果为准。建议在重要决策前咨询专业律师。
